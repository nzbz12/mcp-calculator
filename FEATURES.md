# MCP Cloud Studio 功能特性详解

## 🎯 产品概述

**MCP Cloud Studio** 是一个专业的AI工具集成与管理平台，基于 Model Context Protocol (MCP) 构建。平台提供了完整的Web界面，让用户能够轻松上传、部署、管理和监控自定义的MCP工具。

## 🏗️ 技术架构

### 后端技术栈
- **FastAPI**: 现代化的异步Web框架
- **SQLAlchemy**: ORM数据库操作
- **SQLite**: 轻量级数据库
- **WebSocket**: 实时双向通信
- **JWT**: 安全身份验证
- **Pydantic**: 数据验证和序列化

### 前端技术栈
- **Bootstrap 5**: 响应式UI框架
- **Vanilla JavaScript**: 现代ES6+语法
- **Axios**: HTTP客户端
- **WebSocket API**: 实时通信

## 🔐 安全特性

### 用户认证系统
- **JWT令牌认证**: 无状态的安全认证机制
- **密码哈希**: 使用bcrypt安全存储密码
- **图形验证码**: 防止自动化攻击
- **会话管理**: 自动过期和刷新机制
- **权限控制**: 用户和管理员角色分离

### 数据安全
- **输入验证**: 严格的数据格式验证
- **文件类型检查**: 限制允许的文件格式
- **文件大小限制**: 防止存储滥用
- **路径安全**: 防止目录遍历攻击
- **SQL注入防护**: ORM层面的安全保护

## 📁 文件管理系统

### 上传功能
- **拖拽上传**: 直观的文件拖拽界面
- **多格式支持**: .py, .txt, .json, .yaml, .yml
- **实时预览**: 上传前文件内容预览
- **进度显示**: 上传进度实时反馈
- **错误处理**: 详细的错误信息提示

### 存储管理
- **用户隔离**: 每用户独立的文件空间
- **唯一命名**: UUID确保文件名唯一性
- **元数据记录**: 完整的文件信息追踪
- **自动清理**: 删除工具时自动清理文件

## ⚡ MCP工具管理

### 工具生命周期
1. **创建**: 上传文件并配置基本信息
2. **部署**: 自动创建工作环境
3. **启动**: 一键启动工具进程
4. **监控**: 实时状态和性能监控
5. **停止**: 安全停止工具进程
6. **删除**: 完整清理工具和文件

### 进程管理
- **自动化启停**: 简单的一键操作
- **进程隔离**: 每个工具独立的进程空间
- **资源监控**: CPU、内存使用情况追踪
- **异常恢复**: 自动重启机制
- **优雅关闭**: 安全的进程终止流程

### WebSocket集成
- **实时通信**: 与MCP端点的实时数据交换
- **自动重连**: 网络中断后自动恢复连接
- **状态同步**: 实时同步工具运行状态
- **错误处理**: 连接异常的自动处理
